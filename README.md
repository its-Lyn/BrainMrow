BrainMrow is a derivative of the Brainfuck language, Meowified!~ UwU


## Language Specifics:
- All keywords are separated by spaces.

- Language files have .bm extension.

#### Keywords:
```
meow ->  +
hiss ->  -
mrow ->  <
mrrp ->  >
Purr ->  [
purR ->  ]
~    ->  .
```
